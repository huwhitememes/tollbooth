How it works

01

Discover

Your agent fetches /.well-known/agent.json to learn tool names, prices, and the MCP endpoint URL.

02

Call

Agent calls the MCP tool or HTTP endpoint. Server responds with HTTP 402 and an x402 payment challenge.

03

Pay & receive

Agent pays in USDC on Base via x402, retries with the payment header, and receives structured JSON.
